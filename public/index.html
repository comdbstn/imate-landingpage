AI Agent 주요 기능

🎯 자동 고객지원

  • 실시간 채팅 상담
  • FAQ 기반 자동 응답
  • 고객 문의 접수 및 카테고리 분류
  • 고객 문의 접수 → 티켓 생성 → 담당자 할당 → 처리 상황 자동 알림
  • 카카오톡, 웹채팅 등 연동 가능

📨 이메일 및 알림 자동화

  • 수신 메일 내용 분석
  • 키워드 기반 중요도 판단
  • 관련 담당자에게 자동 알림
  • CRM에 기록

📑 문서 처리 및 보고서 작성

  • 업로드된 문서(PDF, Word 등)의 내용 분석
  • 핵심 요약 자동 추출
  • 계약서 초안 생성 가능
  • 회의록 작성

🧮 데이터 수집·정리

  • 외부 웹/공공데이터/API로부터 정기적 수집
  • 필요한 정보 추출 및 가공
  • Google Sheet, Airtable 등에 정리 저장

📅 일정 및 예약 관리

  • 캘린더(Google, Outlook) 연동
  • 가능한 시간 탐색 → 일정 등록
  • 알림 발송
  • 예약 내용 Google Sheet에 자동 업로드

🔁 반복 업무 자동화

  • 메일/SMS 정기 발송
  • 주기적 리포트 생성 및 전달
  • 서버 상태 모니터링, 반복 점검 자동화

📢 마케팅 콘텐츠 생성

  • SNS용 문구 작성
  • 광고문구 초안 작성
  • SEO 키워드 기반 콘텐츠 자동 작성

🎯 목표 기반 자동 행동

  • 재고 상태 분석 → 부족 시 발주 요청
  • 특정 조건 만족 시 자동 리마인드, 제안, 쿠폰 발급
  • 조건 기반 트리거 설정 가능

👤 고객 맞춤형 서비스

  • 고객 구매 이력/장바구니 분석
  • 맞춤 할인 쿠폰 발급
  • 개인화된 응대 및 마케팅 대응

우리의 AI 캐릭터들

지

지유 - 고객 상담 매니저

실시간 채팅, FAQ 응답, 문의 분류를 통해 고객 지원을 자동화합니다.

하

하린 - 정보 리서처

웹 검색, 문서 요약, 뉴스 수집 등 필요한 정보를 빠르게 찾아 정리합니다.

수

수아 - 일정 도우미

캘린더 연동, 예약 관리, 회의록 작성을 통해 일정 관리를 돕습니다.

준

준호 - 콘텐츠 제작자

SNS 문구, 광고 카피, SEO 키워드 기반 콘텐츠 생성을 지원합니다.

소

소연 - 문서 요약 정리

회의록, 리포트 요약 및 계약서 초안 작성을 통해 문서 작업을 효율화합니다.

윤

윤구 - 반복업무 자동화

데이터 수집·정리, 정기 발송 등 반복적인 업무를 자동으로 처리합니다.

다양한 시스템과 유연하게 연동

상담/응답

KakaoTalk 채널, Webchat, CRM API

문서 요약

PDF, .docx, 텍스트 업로드 → 요약

일정 관리

Google Calendar, Outlook API

예약 데이터

Google Sheets, Airtable

발송

이메일 (SMTP, Mailgun), SMS (Twilio)

외부 시스템

ERP, 쇼핑몰 API, Slack, Notion

다양한 산업 분야에서의 활용

고객문의 자동처리

카카오 문의 → 분류 → 티켓 생성 → 고객 자동 알림

회의요약 자동화

회의록 업로드 → 요약 정리 → 요약본 팀에 전달

재고 자동 발주

재고 수량 모니터링 → 부족 시 공급업체 발주

예약 관리

고객 요청 → 가능한 시간 확인 → 일정 등록 및 알림

마케팅 캠페인 콘텐츠

제품 키워드 입력 → SNS 카피/광고 문구 생성

리서치 보조

키워드 검색 → 뉴스/문서 수집 → 요약 제공
